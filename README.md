# Allinian-Social-Account-Tool
This tool is for defensive security, education, and awareness only.
Do NOT use it against real accounts, platforms, or users without explicit authorization.

This protects you legally.


OK This is how to INSTALL the Allinian Social Account Tool
(A) 1. SYSTEM REQUIREMENTS
 Python 3.8+

 Linux (Kali, Ubuntu), macOS, or Windows

 Terminal / Command Prompt

2. Check Python:
  python3 --version
Allinian-Social-Account-Tool/
│
├── allinian_social_account_tool.py
├── logger.py
├── allinian_security.log   (auto-created)
└── README.md

3. Download / clone the tool
4. 
git clone https://github.com/yourusername/Allinian-Social-Account-Tool.git

cd Allinian-Social-Account-Tool

(B) Option B:Manual setup
1.Create a new folder
2.Paste 
      ~main tool code - allinian_social_account_tool.py
      ~logging code- logger.py or logs.py


4.Run the tool

From inside the folder:

python3 allinian_social_account_tool.py


5. If Python 3 is default:

python allinian_social_account_tool.py


You should see the pink banner and the menu panel.

6. Logs location

Logs are automatically saved to:

allinian_security.log


Every action (login attempts, audits, phishing scan, etc.) is recorded.

7. You can view logs:

cat allinian_security.log


Or live:

tail -f allinian_security.log

 Tested environments

✔ Kali Linux
✔ Ubuntu
✔ macOS
✔ Windows (PowerShell / CMD)

8. Optional: Make it executable (Linux/Kali)
   
chmod +x allinian_social_account_tool.py
./allinian_social_account_tool.py



No root access required.
